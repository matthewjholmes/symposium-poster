# Symposium poster landing page

This folder is ready to publish as a simple GitHub Pages project site.

## Temporary repository and URL

Repository: `symposium-poster`

Default Pages URL: `https://matthewjholmes.github.io/symposium-poster/`

## Replace the placeholders

1. Replace `poster.pdf` with the real poster PDF, preserving the filename.
2. Edit the title, authors, event details, abstract, citation, and contact line in `index.html`.
3. Update `citation.bib` and `citation.ris`.

## Publish through the GitHub website

1. Create a new public repository named `symposium-poster` under `matthewjholmes`.
2. Upload every file and folder in this package to the repository root.
3. Open **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch **main** and folder **/(root)**, then save.
6. After deployment completes, open `https://matthewjholmes.github.io/symposium-poster/`.

## Rename later

Renaming the repository changes the default project-site URL. Update the QR code only after choosing the final repository name, unless a custom domain or redirect is added first.
